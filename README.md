# StudyBuddy

**Dicoding Capstone Project (Work in Progress)**

## Overview
StudyBuddy is a web-based study companion application built with vanilla HTML, CSS, and JavaScript. It provides user authentication and a dashboard for study management.

## Features
- User registration (signup.html)
- User login (login.html)
- Main dashboard (dashboard.html and index.html)
- Responsive styling (style.css)
- Core functionality (script.js)

## Quick Start
1. Clone the repo: `git clone https://github.com/RafelianAsura/StuddyBuddy.git`
2. Open `index.html` or `dashboard.html` in your browser.
3. No additional setup required!

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
